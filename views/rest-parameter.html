ES6: Use the Rest Parameter with Function Parameters

Enlace

Se usa para crear funciones mas flexibles, con el parametro rest se pueden crear funciones que reciban un numero variable de parametros. Estos argumentos son almacenados en un array que se accede desde dentro de la funcion. 

        function howMany(...args) {
            return "You have passed " + args.length + " arguments.";
            }
            console.log(howMany(0, 1, 2)); // You have passed 3 arguments.
            console.log(howMany("string", null, [1, 2, 3], { })); // You have passed 4 arguments.

...args elimina la necesidad de comprobar el array y nos permite usar filter, map, y reduce
