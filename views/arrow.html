ES6: Uso de funciones Arrow para escribir funciones anonimas

Funcion anonima

        const myFunc = function() {
            const myVar = "value";
            return myVar;
            }

Funcion Arrow

        const myFunc = () => {
            const myVar = "value";
            return myVar;
            }

Funcion Arrow - Si no hay body y solo return , se puede omitar la palabra return y los parentesis

        const myFunc = () => "value";

ES6: Write Arrow Functions with Parameters

Enlace

Se pueden pasar parametros como a una funcion normal.

        const doubler = (item) => item * 2;

Si solo hay un parametro se pueden omitir los parentesis.

        const doubler = item => item * 2;

Se puede pasar mas de una parametro a la funcion arrow

        const multiplier = (item, multi) => item * multi;
