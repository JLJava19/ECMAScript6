ES6: Use the Spread Operator to Evaluate Arrays In-Place


Enlace

ES6 introduce un nuevo operador: spread, nos permite expandir Arrays y otras expresiones en lugares donde multiples parametros son esperados

El codigo ES5 escrito abajo usa apply() para computar el valos maximo en un array:

        var arr = [6, 89, 3, 45];
        var maximus = Math.max.apply(null, arr); // returns 89

Teniamos que usar Math.max.apply(null, arr) porque max(arr) devuelve NaN. Math.max() espera argumentos separatos por coma, pero NO UN ARRAY. El operador spread hace la sintaxis mucho mejor de leer y mantener. 

        const arr = [6, 89, 3, 45];
        const maximus = Math.max(...arr); // returns 89

...arr devuelve un unpacked array. Extiende el array. Sin embargo, el operador de propagación solo funciona en el lugar, como en un argumento para una función o en una matriz literal. El siguiente codigo no funcionara 

        const spreaded = ...arr; // will throw a syntax error

Solucion

        const arr1 = ["JAN", "FEB", "MAR", "APR", "MAY"];
        let arr2;

        (function() {
            "use strict";
            arr2 = [...arr1]; // change this line
        })();

        console.log(arr2);

Explicacion del código

Unpacking the arr1 using the spread operator and then copying those values to arr2
